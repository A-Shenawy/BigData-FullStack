# 🚀 Real-Time Analytics on Healthcare Data Using MIMIC-III  

This project implements an **end-to-end Big Data pipeline** for real-time analytics on **MIMIC-III**, a **critical care dataset**. It leverages **Apache Spark, Kafka, and Hadoop** for **data processing**, **SQL & NoSQL databases** for storage, and **machine learning models** for predictive insights.

## **📌 Project Overview**  
🚑 **MIMIC-III** (**Medical Information Mart for Intensive Care**) is a database containing de-identified health data from **critical care patients**. This project builds a **real-time analytics platform** to analyze patient data, detect anomalies, and generate predictive insights.  

### **🎯 Key Objectives:**  
✅ **Stream healthcare data** in real-time using **Apache Kafka**  
✅ **Process large-scale patient records** with **Apache Spark & Hadoop**  
✅ **Store structured & unstructured data** in **SQL & NoSQL databases**  
✅ **Train predictive ML models** for risk assessment & ICU monitoring  
✅ **Visualize insights** using dashboards (Power BI / Tableau)  

---

## **🛠️ Technologies Used**  
📌 **Big Data Tools:** Apache Spark, Hadoop, Apache Kafka  
📌 **Databases:** PostgreSQL, Cassandra, HDFS  
📌 **Machine Learning:** Scikit-Learn, TensorFlow, PySpark ML  
📌 **Visualization:** Tableau, Power BI, Matplotlib  
📌 **Deployment:** Docker, Kubernetes  

---

## **📊 Data Processing Pipeline**  
### **1️⃣ Data Ingestion & Streaming**  
🔹 **Apache Kafka**: Ingest real-time patient vitals & ICU logs  
🔹 **Batch & Streaming Processing**: Hybrid approach for historical + real-time analysis  

### **2️⃣ Data Storage & Management**  
🔹 **HDFS & Cassandra**: Scalable storage for structured & unstructured data  
🔹 **PostgreSQL**: Relational database for structured analytics  

### **3️⃣ Machine Learning & Predictive Analytics**  
🔹 **Predicting patient risk levels** based on real-time vitals  
🔹 **Anomaly detection in ICU trends**  
🔹 **Time-series forecasting** for patient deterioration patterns  

### **4️⃣ Data Visualization & Insights**  
🔹 **Power BI & Tableau Dashboards** for hospital staff  
🔹 **Automated alerts** for critical conditions
